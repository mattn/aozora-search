# aozora-search

青空文庫検索

## Usage

芥川龍之介の作品を収集
```
$ aozora-collector
```

芥川龍之介の作品を検索
```
$ aozora-search
```

## Installation

```
$ go install github.com/mattn/aozora-search/cmd/aozora-collector@latest
```

```
$ go install github.com/mattn/aozora-search/cmd/aozora-search@latest
```
